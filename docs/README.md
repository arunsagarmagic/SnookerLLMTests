# Snooker LLM Tests Results

This folder contains static HTML webpages generated by different LLMs as part of a single-shot testing experiment for creating a Snooker scoring webapp.

## Available Results

- **Kimi 2.6** - `Kimi2.6.html`
- **ChatGPT Free** - `chatgpt.free.html`
- **Claude Sonnet 4.6** - `claude.sonnet4.6.html`
- **Cohere North Mini** - `cohere-north-mini-code.html`
- **DeepSeek Expert** - `deepseek.expert.html`
- **Gemma 3.1B** - `gemma31b.html`
- **GLM 5.2** - `glm5.2.html`
- **Nemotron 120B** - `nemotron-3-super-120b-a12b.html` and `.fixed.html`
- **OpenAI GPT OSS 120B** - `openaigpt-oss-120b.html`
- **OpenAI GPT OSS 20B** - `openaigpt-oss-20b.html`
- **Owl Alpha** - `owlalpha.html`

## Viewing Results

Start with `index.html` for a directory browser, or access individual LLM results directly.
